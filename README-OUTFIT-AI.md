# OUTFIT AI PWA

iPhone・iPad・PCで使えるホーム画面対応Webアプリです。

## 搭載機能

- シャツ・パンツ・靴のマスター登録、編集、削除
- iPhoneのカメラ撮影と写真ライブラリ選択
- 選択直後の写真表示と写真からの色候補
- ブランド、色、種類、各属性の修正
- タグ・靴箱の追加撮影
- 現在地の天気と昨日の天気表示
- シャツ・パンツを選んだ後、おすすめ靴3足を表示
- 「今日これを履く」後の完了画面
- コーデ履歴、使用回数、未使用靴、メンテナンス分析
- D1によるデータ保存、R2による写真保存
- PWAマニフェストとホーム画面対応

## iPhoneで使う

公開URLをSafariで開き、「共有」→「ホーム画面に追加」を選びます。
カメラ・現在地は、初回利用時に許可してください。
